# Uppgift-3-FB-W

Du kan se det färdiga resultatet här:

https://runegranath.github.io/Uppgift-3-FB-W/

Tekniker:

Vite – Byggverktyg för snabb utveckling.

Sass (SCSS) – Avancerad styling med:

Mixins & If/Else: För dynamiska knappar (Primary/Secondary).

Variabler: Centraliserade färger för Light/Dark mode.

@extend: Återanvändbar kod för bildtexter och layout.

GitHub Actions – Automatisk publicering (CI/CD) vid varje push.

Funktioner:

Light/Dark Mode: Anpassar sig automatiskt efter systeminställningar.

Animationer: Hover-effekter med glidande gradientskiften.
