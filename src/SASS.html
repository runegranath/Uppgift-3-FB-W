Utvärdering av arbetet

Fördelarna med SASS är att det gör kodningen mer strukturerad för större projekt. Det börjar bli tydligare vid det här laget hur man faktiskt gör i praktiken vid större jobb och fler personer är inblandade och varför det behöver struktureras på det här viset.
Just nesting till CSS var något jag intuitivt saknat tidigare och stört mig på, så att den möjligheten finns gör det mycket smidigare. Men hade man sedan gått direkt på SASS istället för att i början bara kört med CSS kunde det blivit lite väl mastigt.
Vid mindre arbeten av simplare typ har bara CSS kunnat varit en fördel över SCSS för att jobba snabbare och slippa strukturera nya filer. Speciellt när jag nu lärt mig att vanlig CSS också har stöd för nesting sedan ett tag även om det kan bli problem med äldre webbläsare. Annars är det lätt att se fördelarna med SASS när det kommer till större och mer komplicerade projekt där användandet av variabler och funktioner kan göra det mycket smidigare. Annars är det allt som oftast kul att greja med olika CSSformer, iallafall när elementen beter sig som de ska till slut.
